Pymodoro by jdsolucoes

Sistema em Python/GTK para gerenciamento de tempo usando a técnica de POMODORO

Ola, seja bem vindo ao site do Pymodoro, aqui você podera efetuar o download do software, contribuir para o seu crescimento e se beneficiar das inumeras funções que o Pymodoro oferece, baixe, utilize e se puder desenvolva. obrigado.

Dependencies

Bindings do Mysql, Sqlite, Pango,Gtk,alsa etc...

Install

Só executar o Pymodoro.py
